To run the game:

1.) Download xampp (https://www.apachefriends.org/index.html)

2.) In your xampp directory, place this entire project in xampp's htdocs folder.

3.) Start xampp, then run "Apache" and "MySQL"

4.) In your browser, go to localhost/phpmyadmin and upload the SQL file from the project's "SQL File" folder (playerinfo.sql)

5.) To play(copy this to the url ):
	localhost/LavyrinthosDEFENSE/index.php

6.) Game starts.
